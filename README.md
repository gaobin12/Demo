Hello Git

第一次练习git bash 命令

复习git上传步骤

git init 初始化

git add * 纳入缓存

git status 查看状态

git commit -m "" 纳入版本库

git remote add origin https://github/gaobin12/demo 建立连接

git remote -v 查看连接

git pull origin master 拉取线上的数据

在sublime中修改 

git add * 将修改后的纳入缓存

git commit -a -m "" 纳入版本库

git push -u origin master 上传
